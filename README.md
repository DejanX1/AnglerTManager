# AnglerTManager
A Java-based desktop application for managing team-based angling competitions on lakes.

## Project Overview
This system is engineered for real-world practical application in sport fishing. It solves the complexity of organizing team tournaments by automating the drawing process and providing instant, live leaderboards based on catch data entered directly from the field.

## Key Features
Real-Time Catch Logging: Enables instant recording of catches (species and weight) during live competition;
Instant Live Standings: Automatically calculates and updates rankings for both individual competitors and teams the moment a new catch is registered;
Tournament History & Archive: Provides full insights into all past competitions, including statistics and historical performance data;
Dynamic Event Setup: Comprehensive management of new tournaments, including the assignment of teams, lakes, judges, and specific fishing methods;
Fair Play Engine: Automatically distributes team members across different sectors (three-sector system) to eliminate the "luck of the draw" regarding fishing spot quality;
Catalog Management: Built-in dictionaries for managing various fish species and diverse fishing techniques;
